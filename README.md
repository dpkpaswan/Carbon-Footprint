# Carbon-Footprint

**CarbonLens** — A Carbon Footprint Awareness Platform built with React + Vite.

Calculate your carbon footprint, track your progress with daily streaks, and discover personalized actions to reduce your environmental impact.

## Features

- 🧮 **Carbon Calculator** — Multi-step form covering Transport, Food, Energy, and Shopping
- 📊 **Dashboard** — Interactive charts (Recharts) comparing your footprint vs global averages
- 🎯 **Action Recommendations** — Personalized tips sorted by your highest-emitting category
- 🔥 **Streak Tracker** — Daily check-in with a GitHub-style contribution grid

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- Recharts
- Vitest + Testing Library
- localStorage for persistence

## Getting Started

```bash
npm install
npm run dev
```
